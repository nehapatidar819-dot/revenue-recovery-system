# revenue-recovery-system
# Revenue Recovery & Churn Prediction System

This project analyzes customer behavior and predicts churn using machine learning. 
It also identifies opportunities to recover revenue using customer segmentation, 
usage patterns, complaints, and purchase history.

## Key Features
- Customer segmentation (High value, At-risk, Recoverable)
- Churn probability prediction
- Revenue opportunity detection
- Behavior analytics (usage, complaints, refunds)
- Business recommendations
- Optional: Interactive dashboard (Streamlit)

## Files Included
- customer_churn_data.csv — mock data for analysis
- notebooks — data cleaning, EDA, model training, recommendations
- dashboard/app.py — Streamlit dashboard
- requirements.txt — dependencies needed for the project

## Tools Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn

## How to Use
1. Clone the repo  
2. Install dependencies using `pip install -r requirements.txt`  
3. Run the Streamlit dashboard using:  
   `streamlit run dashboard/app.py`
